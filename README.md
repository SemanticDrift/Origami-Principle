## What This Is

The Origami Principle demonstrates that circular geometry can be expressed exactly using rational fractions derived from discrete folding operations. By treating a circle as the folded state of a square, the relationship C = (22/7)D is derived through harmonic principles, eliminating the need for transcendental constants within a construction-based framework.

π is not wrong. It is unnecessary.

## The Problem It Solves

Modern geometry treats π as a required primitive. π is transcendental: it cannot be expressed as a ratio of integers and its decimal expansion never terminates or repeats. Every calculation involving π introduces approximation. Repeated application amplifies error.

The Origami Principle shows that this approximation is not inherent to circular geometry. It is an artifact of imposing continuous measurement on discrete harmonic structure. Within a construction-based framework, exact rational arithmetic is not only possible but structurally natural.

## The Derivation

**Step 1: The Square-Circle Construction**

Begin with a unit square of side length D = 1. Inscribe a circle whose diameter equals the side of the square. Divide the square into eight congruent right isosceles triangles. The portions extending beyond the circle represent the unfolded material required for the folding operation that produces circular symmetry.

**Step 2: Baseline Folded Circumference**

In a discrete folding model, circumference emerges from the perimeter of the folded square state:

```
C₀ = 3D
```

This is the zero-order result, representing bulk geometry of circular closure. The factor 3 arises from harmonic closure through the n = 3 torsion shell and the three independent axes required for rotational symmetry in physical space.

**Step 3: Torsion Adjustment**

The baseline C₀ = 3D does not complete circular closure. From the Law of Admissibility R = 4, the square imposes quartering symmetry. The n = 3 torsion shell introduces the 1/6 constant. Their interaction yields a residual first-order correction:

```
ΔC = D/7
```

**Step 4: Final Expression**

```
C = 3D + D/7 = D(3 + 1/7) = (22/7)D
```

This is the rational circumference formula derived from discrete folding principles.
